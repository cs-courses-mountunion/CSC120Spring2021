Multiple Choice Question # 5:

What are the smallest and largest possible values that will be stored in the object num when these statements are executed? 
    Random randGenerator = new Random();
    Integer num = randGenerator.nextInt(78);
smallest is 0 and largest is 78
smallest is 1 and largest is 78
smallest is 0 and largest is 77
smallest is 1 and largest is 77
smallest is 0 and largest is 79
smallest is 1 and largest is 79
None of these
The parameter passed to the nextInt method of the Random class represents the number of possible values that will be returned by the method.  The smallest value is 0, and therefore the 78th value would be 77.  The largest value returned by the method is always one less than the parameter value.
