Multiple Choice Question # 6:

What are the smallest and largest possible values that will be stored in the object number when these statements are executed? 
    Random randGenerator = new Random();
    Integer number = randGenerator.nextInt(200) + 100;
smallest is 0 and largest is 299
smallest is 100 and largest is 199
smallest is 100 and largest is 299
smallest is 200 and largest is 299
None of these
The nextInt method of the Random class returns a random Integer value that is greater than or equal to 0 and is strictly less than the parameter that is passed to the method, so if we add 100 to a random Integer in the range from 0 to 199, the smallest possible value is 0 + 100 == 100, and the largest possible value is 199 + 100 == 299.
